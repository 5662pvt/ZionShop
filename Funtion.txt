8. CÁCH CLAUDE CODE NÊN HIỂU PROJECT

Bạn cần file:

CLAUDE.md

Đây là “system prompt” cho toàn bộ project.

Ví dụ:

# Architecture Rules

- Project uses Modular Monolith + Clean Architecture
- No direct Infrastructure access from API
- All business logic must stay in Application or Domain
- Use CQRS pattern
- Use MediatR
- Use Repository Pattern
- Use Domain Events
- Use Result Pattern
- Use FluentValidation

# Coding Rules

- Async all the way
- No fat controller
- No business logic in controller
- No static service
- No circular dependency

# Module Rules

- Modules communicate using Events
- No direct DbContext access across modules

Cái này cực kỳ quan trọng.

❌ shared state lung tung
❌ business logic trong UI
❌ hardcode permission
